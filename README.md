# Instagram Reels Downloader (using Chrome Login Session)

This Python tool downloads **Instagram Reels** from any public profile (or private profiles you already follow).  
You **do NOT need to enter your password** ‚Äî it uses your **Chrome browser session** to authenticate safely.

---

## Features
- Uses **Chrome session cookies** ‚Üí No password required
- Downloads **only Reels** (video posts)
- Works with **public & private accounts** (if you're already following)
- Automatically saves reels into a folder: `<username>_reels/`
